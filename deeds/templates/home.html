Getting Started

A quick guide to using deeds

This guide is here to make the first few steps feel obvious, while also showing you the extra features you can use when you want them.

1. Start where you are

If you are not ready to commit, use contemplated items on the dashboard as a holding space. If you already know what matters, create a goal straight away.

2. Break goals down

Add steps in the step planner and make them small enough to feel clear and achievable. You can schedule them for past, current, or future days.

3. Complete and reflect

When a step is done, mark it complete. You can add a reflection straight away, or later if you want to capture how it felt and what you learned.

4. Logging is optional

Some steps can also be tracked as activities with time, distance, sets, reps, and notes. If you did not set this up at first, you can still keep using simple steps without logging.

5. Archive and revive goals

When all steps are done, archive the goal as achieved. If you later want to add more steps, mark it as not achieved again and continue.

6. Use the app your way

You can focus only on goals and steps, only on activities, or use both together. deeds is meant to support momentum, not force one rigid workflow.
